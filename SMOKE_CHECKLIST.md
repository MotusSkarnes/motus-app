# Smoke Checklist

Bruk denne sjekklisten etter endringer i kjerneflyten.

## Trener -> Medlem programflyt

- [ ] Logg inn som trener med ekte konto (ikke demo).
- [ ] Gå til `Klienter` og velg en kunde.
- [ ] Lagre et nytt program på kunden.
- [ ] Bekreft suksessmelding i trenervisning.
- [ ] Logg inn som medlem med samme e-postprofil.
- [ ] Gå til `Mine treningsprogram` og bekreft at programmet vises.

## Økt og feedback

- [ ] Start en økt fra medlemssiden.
- [ ] Fullfør økten og send emoji-feedback + notat.
- [ ] Bekreft at økten vises under `Siste økter` hos medlem.
- [ ] Logg inn som trener og åpne kundens `Økter`-fane.
- [ ] Bekreft at øktdata, feedback og notat er synlig.

## Meldinger og grunnflyt

- [ ] Send melding fra trener til medlem.
- [ ] Les og svar fra medlemssiden.
- [ ] Bekreft at begge meldinger vises i begge visninger.

## Rask driftssjekk

- [ ] Sjekk `Admin` -> `Driftsstatus` for backend/session-status.
- [ ] Hvis "Demo (lokal)" vises: ikke bruk dette til å validere synk.
